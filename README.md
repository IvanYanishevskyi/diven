# Diven

Diven is a project by @IvanYanishevskyi. As it is currently under development, this README provides a basic overview and setup instructions.

## About The Project

Diven is an AI-powered data analysis tool designed to transform your spreadsheets into actionable insights. Upload your data, ask questions in natural language, and receive instant visualizations and analysis. Diven uses a powerful AI to understand your queries and automatically generates charts and graphs to help you discover trends and patterns in your data. This allows you to focus on insights, not on complex formulas or code.
This project is built with Python.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Python 3.x
*   pip

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/IvanYanishevskyi/diven.git
    ```
2.  Navigate to the project directory
    ```sh
    cd diven
    ```
3.  Install Python packages
    ```sh
    pip install -r requirements.txt
    ```
4. Create a `.env` file in the root directory and add the necessary environment variables. Refer to the `.env.example` if available, or start with the following based on the project structure:
    ```
    # Example .env file
    DEBUG=True
    SECRET_KEY='your-secret-key'
    ```
